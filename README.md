## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Setup New Project

```bash
# Nest JS Project Create
$ npm i -g @nestjs/cli
$ nest new project-name

# Nest Js Resource
$ nest g resource customer

# Install Project Nosql(MongoDB) 
$ npm i @nestjs/mongoose mongoose

# Install Project to dotenv
$ npm i dotenv

# Crate Project to (src\customer>) customer.models.ts
$ touch customer.models.ts
````

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
